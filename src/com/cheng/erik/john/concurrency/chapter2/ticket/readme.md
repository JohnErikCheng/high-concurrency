### 第一版本
```
 - 实现方式:
    1、定义一个线程类extends Thread类，完成其中的实现逻辑
    2、定义一个实现类，创建多个线程，实现线程类，并start()
 - 出现的问题
    1、多个实现线程各自实现自身的线程，行成各玩各的状态
```
